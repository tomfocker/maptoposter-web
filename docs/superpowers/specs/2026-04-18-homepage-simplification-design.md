# Homepage Simplification Design

## Goal

进一步压缩首页噪音，移除折叠式高级区，把主题总览改成页面内弹窗，并继续删掉首屏不必要的说明文字。

## Scope

本轮只调整首页模板、主题交互和相关样式，不改后端接口、不改海报生成流程、不改主题数据源。

## Approved Direction

- 不再使用折叠式“精修面板”
- 左侧表单改为常驻分组卡片
- 主题总览不常驻页面，改为页面内部弹窗
- 顶部和首屏的辅助说明继续减到最少

## Layout

首页左侧表单改为五个常驻分组，顺序固定：

1. `地点`
2. `主题`
3. `输出设置`
4. `文字覆盖`
5. `构图微调`

其中：

- `地点` 保留城市和国家/省份输入
- `主题` 保留主题下拉、当前主题色带、查看全部主题按钮
- `输出设置` 保留尺寸、DPI、导出格式
- `文字覆盖` 保留标题、副标题、文案语言
- `构图微调` 保留地图范围、水平偏移、垂直偏移、道路粗细缩放

## Text Reduction

本轮继续去掉这些文案：

- 顶栏右侧说明
- 左侧主标题下的辅助说明
- 预览区右上角的格式说明
- 折叠相关文案和状态字样
- 多余的字段解释句

页面中仅保留：

- 必要标题
- 必要字段名
- 少量占位词

## Theme Gallery Modal

主题色板总览改为页面内部弹窗：

- 入口在主题卡片内，按钮文案为“查看全部主题”
- 点击后打开覆盖在当前页面上的内部弹窗
- 弹窗内展示全部主题色板卡片
- 支持点击关闭按钮和遮罩关闭
- 当前页面主内容不再直接展示全部主题色板

## Interaction

- 删除折叠按钮和展开/收起逻辑
- 新增主题弹窗的打开/关闭逻辑
- 当前主题切换时，首屏色带预览继续联动更新

## Testing

补模板约束测试，覆盖以下行为：

- 页面不再出现折叠面板与切换按钮
- 输出设置、文字覆盖、构图微调为常驻分组
- 主题总览位于弹窗容器内，而不是主页面常驻区
- 顶部与首屏指定文案被移除
